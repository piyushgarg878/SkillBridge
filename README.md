# SkillBridge

https://github.com/user-attachments/assets/9d5576f4-c6ba-4734-b88c-99582b9da7ce

SkillBridge is a modern job application platform designed to bridge the gap between talent and opportunity. It streamlines the hiring process for both recruiters and candidates through intelligent features and a user-friendly interface.

## Key Features

*   **For Candidates:**
    *   Browse and apply for jobs with a single click.
    *   AI-powered resume scoring to match your profile with job requirements.
    *   Track application status in real-time.
    *   Premium dashboard to manage your career journey.

*   **For Recruiters:**
    *   Post and manage job listings.
    *   View applicant summaries and AI-generated scores.
    *   Efficiently shortlist candidates.

## Tech Stack

*   **Frontend:** Next.js 15, Tailwind CSS, Shadcn UI
*   **Backend:** Next.js API Routes, Supabase (PostgreSQL & Storage)
*   **AI/ML:** Python (FastAPI) for resume scoring
*   **Authentication:** NextAuth.js

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables.
4.  Run the development server: `npm run dev`
